# 竞赛/培训标准库封装工具

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

一个专为竞赛、培训选拔场景设计的标准库二次封装工具——核心目标是帮开发者减少基础功能的重复编码时间，避免因适配、调试基础逻辑耽误核心任务，让你更专注于竞赛/考核的核心需求。

## 核心定位
- 基于官方标准库封装，**不修改底层逻辑**，保留原标准库的稳定性与兼容性
- 简化高频场景的调用流程，减少冗余代码，提升开发效率
- 无额外第三方依赖，仅依赖原生标准库，适配竞赛/培训的环境要求

## 适用场景
- 编程竞赛、算法竞赛、开发类竞赛的快速开发
- 技术培训、选拔考核中的项目实战与任务落地
- 短期开发场景中需要快速实现基础功能、聚焦核心逻辑的需求


## 后续更新
- 采用 **不定时更新** 机制，根据实际使用反馈与场景需求迭代
- 优先补充竞赛/培训中高频出现的功能封装
- 持续修复适配问题，优化调用体验
- 不新增复杂特性，保持轻量、高效的核心定位

## 注意事项
1. 本项目仅为标准库的「包装工具」，不涉及底层技术改造，核心能力依赖官方标准库
2. 使用前请确认目标竞赛/培训允许使用对应标准库（封装不改变原库性质）
3. 支持基于本项目二次开发，可根据自身需求灵活调整封装逻辑
4. 若发现bug或有场景需求，欢迎提交issue或PR
